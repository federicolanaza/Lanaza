# **App Name**: LibreConnect Analytics

## Core Features:

- Secure Google Authentication: Allow users to log in securely using Google accounts, restricted to institutional email domains as specified.
- User Role Management: Implement two distinct user roles: 'Admin' with full access and 'Visitor' with restricted check-in capabilities.
- Visitor Check-in Form: Provide a clean form for visitors to input 'College Department' and 'Reason for Visit' upon login.
- Dynamic Welcome Notification: Display a 'Welcome to NEU Library!' notification after a visitor successfully submits their check-in details.
- Real-time Admin Statistics: An administrative dashboard featuring 'Stat Cards' displaying total visitor counts for the Day, Week, and Month, updated in real-time.
- User Management Panel: Admin interface to search for specific users by name and manage their access with a 'Block User' function.
- AI Visit Trend Analysis Tool: An AI-powered tool for admins to generate insights and summaries on library visitor trends, based on historical check-in data.

## Style Guidelines:

- Primary color: A deep, professional blue (#2E5CB8), symbolizing knowledge and tranquility for a library setting.
- Background color: A very light, almost imperceptibly blue-tinted gray (#F0F1F5) for a clean, open feel.
- Accent color: A rich violet-blue (#1430B8) to highlight interactive elements and draw attention without being overwhelming.
- All text will use 'Inter' (sans-serif), chosen for its modern, objective, and highly legible characteristics suitable for data display and professional content.
- Utilize a consistent set of minimalist line-art icons that enhance readability and intuitively represent various actions and data points.
- Employ a clean, modular layout with clear separation between the visitor interface and the administrative dashboard, prioritizing responsiveness for various screen sizes.
- Implement subtle transition animations for state changes and data loading, providing smooth user feedback without distraction.
