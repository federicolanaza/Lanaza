📚 VirtuLab

A Library Visitor Management & Real-Time Analytics Dashboard for New Era University.

VirtuLab is a web-based application designed to simplify the library check-in process while providing administrators with real-time insights into visitor activity, usage trends, and system monitoring.

🚀 Features

Institutional Login: Only @neu.edu.ph emails allowed

Quick Check-in: Email + Department + Reason + User Type

Real-time Dashboard: Live visitor stats (Day/Week/Month)

Filtering System: By department, reason, and user type

User Management: Search and block users

Active Sessions: Real-time logged-in users

🛠️ Tech Stack

Frontend: TypeScript, Next.js, React

UI: Tailwind CSS, ShadCN UI

Backend: Firebase Firestore

Auth: Firebase Authentication

Hosting: Firebase App Hosting
